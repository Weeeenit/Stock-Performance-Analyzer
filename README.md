# Stock Performance Analyzer

A Streamlit dashboard for analyzing historical performance of Indian stocks and market indices.

## Features

- Overview Dashboard
- Price Analysis
- Return Analysis
- Risk Analysis
- Investment Simulator

## Assets

- NIFTY 50
- SENSEX
- India VIX
- HDFC Bank
- ICICI Bank
- Infosys
- Reliance
- TCS

## Technologies

- Python
- Streamlit
- Pandas
- Plotly
- NumPy

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/Stock-Performance-Analyzer.git

cd Stock-Performance-Analyzer

pip install -r requirements.txt

streamlit run app.py
```
